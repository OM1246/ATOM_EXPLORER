## üöÄ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/interactive-atom-explorer.git
   ```

2. Navigate to the project directory:
   ```bash
   cd interactive-atom-explorer
   ```

3. Open `Interactive_Atom_Explorer.html` in your web browser

### Usage
1. Adjust the number of protons, neutrons, and electrons using the +/- buttons
2. Observe how the atom visualization changes in real-time
3. Click on different UI elements to get detailed information
4. Use the AI assistant for deeper learning
5. Save your favorite atom configurations
